# Typography

For the sake of speed and simplicity, typography components do not exist. Instead the app relies on BEM style class names and HTML attributes.

## Headings

Headings, h1-h6, can be styled using their native HTML tag (e.g. `<h1 />`) or through the use of classNames (e.g. `<div className="h1" />`)

  <h1>H1 element</h1>
</Canvas>
